# everybody.codes

These are my humble (and sometimes slow and messy) solutions to the excellent
https://everybody.codes puzzles, which I really recommend, especially if you
like ducks. 

So far, my solutions are all doing the best I can in R. As with AoC, I haven't
included any of the inputs provided to me (go and download your own), nor any
of the test text, although I did often use the tests to get my answers working.
